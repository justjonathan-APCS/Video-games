Video Games : Corgis Database

The Corgis data bases is filled with valueable information and all are from reliable sources aswell. This information includes...

Features.Max Players : The Features.Max Players is the key that represents the maximum amount of players that are allowed to play the game at the same time.
Features.Multiplatform? : Features.Multi Platform is the key that represents whether or not the game is available on different platforms so that there is no console bias.
Features.Online? : Features.Online is the key that represents that this game is able to support online play. This is a very important one because many people enjoy playing online and if the game does not support online play many people might not buy it.
Metadata.Publishers : Metadata.Publishers is the key that tells you the list of publishers that helped create/develop the game.
